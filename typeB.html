Student Category

TYPE B

Motivated learners ready to grow with structured mentorship

Who they are

At the foundational stage—they have completed coursework and built simple projects, but need guidance to apply concepts. Highly motivated, coachable, and hungry to learn.

What they know

- Basic programming (loops, functions) in one language
- Familiar with concepts but need guidance
- Can follow tutorials and write simple code - Minimal Git or professional workflow experience

What they've done

- 1–2 simple projects (calculator, static site, form)
- Coursework assignments
- Code that works for known problems

What you provide

- Structured onboarding with clear learning objectives
- Paired programming and regular check‑ins
- Bite‑sized tasks with teaching‑focused code review
- Patience and mentorship
